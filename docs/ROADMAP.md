# Calculatify 產品路線圖 (24個月規劃)

> 本文件記錄 Calculatify 未來 24 個月的發展目標與執行計畫。
> 總計開發 170+ 個計算器工具，建立完整的工具型網站生態。

---

## 核心策略

**SEO-First + AI SEO 優先**：所有工具開發皆以搜尋流量最大化為首要目標，透過階段式部署建立權威性並逐步擴展至長尾關鍵字。

## 開發規範

### 每個計算器的標準配置

1. **獨立頁面路由**：`/calculators/{tool-name}`
2. **SEO Meta 標籤**：針對搜尋意圖優化的 title/description
3. **結構化資料**：Schema.org JSON-LD 標記
4. **行動優先設計**：Tailwind CSS 響應式佈局
5. **效能優化**：Core Web Vitals 達標
6. **免責聲明**：金融/健康類工具必備

### 風險管理

- **高風險工具**（金融類）：需完整免責聲明、法律審查
- **中風險工具**（健康類）：需醫療免責聲明
- **低風險工具**（數學/轉換）：標準實作即可
